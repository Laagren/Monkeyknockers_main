För att köra spelet krävs Unity och för att se koden behövs Visual studio.

1. Ladda ner Unity, version 2021.3.2f1 och även Visual studio som följer med.
2. Gå in i da336a_Unity_projekt_G25/Assets/Scenes och öppna "StartScreen" med Unity 2021.3.2f1
3. Sedan är det bara att klicka på play-knappen uppe i mitten
